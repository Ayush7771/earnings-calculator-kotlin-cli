# 💰 Earnings Calculator (Kotlin CLI)

A simple command-line application built with **Kotlin** that calculates total earnings and net income based on user-provided expenses.

---

## 🚀 Features
- Displays earnings from multiple items
- Calculates total (gross) income
- Accepts user input for:
  - Staff expenses
  - Other expenses
- Calculates and displays **net income**

---

## 🛠 Tech Stack
- **Language:** Kotlin
- **Build Tool:** Gradle
- **IDE:** IntelliJ IDEA
